## 💻 WORKFLOW COM OPENCODE.IA

### Pré-requisitos
- ✅ VSCode com MCP Pencil instalado
- ✅ OpenCode.ia conectado como MCP provider
- ✅ Projeto Next.js **não** precisa estar criado (PROMPT 0 ajuda)

### Execução
1. Abrir VSCode
2. Abrir MCP Pencil (Pencil ícone)
3. Para cada PROMPT em OPENCODE_PROMPTS.md:
   - Copiar prompt (ex: PROMPT 1 - Hero)
   - Colar no chat do Pencil
   - Deixar rodar até gerar código
   - Copiar saída → novo arquivo em components/
   - Testar com `npm run dev`

### Exemplo Prático
```
1. PROMPT 0 → Setup: next.config.js, package.json, tsconfig
   npm install
   npm run dev

2. PROMPT 1 → Hero.tsx
   Salvar em components/Hero.tsx
   Testar em http://localhost:3000

3. PROMPT 2 → TechStackShowcase.tsx
   Salvar em components/TechStackShowcase.tsx
   Testar

... e assim por diante até PROMPT 9
```
